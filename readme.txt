A modular plugin for the Madara theme, adding points, currency, store, airdrops, and user engagement features.

== Description ==

Ignis Plugin enhances the Madara theme (WP Manga) with a suite of features to boost user engagement and monetization. Key features include:

- **Points System**: Reward users for reading chapters, commenting, logging in, and more.
- **Currency System**: Manage virtual currency (MangaCoin) for in-plugin purchases.
- **Store**: Allow users to spend points or currency on premium content.
- **Airdrops**: Distribute points or currency to users periodically.
- **User Engagement**: Support referrals, bug reporting, and manga requests.
- **Admin Panel**: Centralized control for all module settings.

Requires the WP Manga (Madara-Core) plugin. Configure via the Ignis Control Center in the WordPress admin.

== Installation ==

1. Upload the `ignis-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Ensure WP Manga (Madara-Core) is installed and activated.
4. Configure settings in **Ignis Control** in the admin menu.

== Frequently Asked Questions ==

= Does this plugin work without the Madara theme? =
No, it requires the WP Manga (Madara-Core) plugin, which is part of the Madara theme.

= How do I enable the Points Module? =
Go to **Ignis Control > Settings** and check "Enable Points Module." Configure rules in **Ignis Control > Points**.

= Can I translate the plugin? =
Yes, edit the `languages/ignis-plugin-en_US.po` file and generate the `.mo` file using Poedit.
